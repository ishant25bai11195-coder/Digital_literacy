## 📌 Project Overview

This project represents my work as a Student Digital Ambassador at VIT Bhopal University. The objective was to build a digital literacy portfolio to help students understand professional online presence, safe communication, and cyber security.

Through five structured tasks, I created resources such as an infographic, digital profiles, coding practice, professional communication samples, and a cybercrime awareness case study. All work is organized in this GitHub repository to showcase practical learning and digital skills.

---

## 📚 Module Summaries

### 🔹 [Task 1: Digital Literacy Awareness Infographic](./task-1-presentation/)

Created an infographic using **Canva** covering key topics such as digital literacy basics, useful digital tools, safe internet practices, email etiquette, and building a professional presence. The design presents information in a simple and visually structured format.

---

### 🔹 [Task 2: Student Digital Portfolio](./task-2-portfolio/)

Set up professional profiles on **GitHub**, **LinkedIn**, and **Kaggle**.  
- GitHub is used to manage and showcase coding projects  
- LinkedIn is used for professional networking  
- Kaggle is used to explore data science and machine learning  

---

### 🔹 [Task 3: Coding & Collaboration Platforms](./task-3-platforms/)

Practiced coding by solving a beginner-level problem on **HackerRank**.  
Created a **Digital Literacy Awareness Quiz** using Google Forms and connected it to a response sheet for data collection.

---

### 🔹 [Task 4: Email Etiquette & Communication](./task-4-email-etiquette/)

Developed professional email writing skills by creating:
- Assignment extension request email  
- Internship application email  

Also created a **Social Media Do’s & Don’ts Checklist** to promote responsible online behavior.

---

### 🔹 [Task 5: Cybercrime Awareness](./task-5-cybercrime/)

Worked on **UPI / Online Payment Fraud** as a case study.  
Explained how scams happen (QR codes, fake requests) and created a **prevention checklist** with practical safety tips for students.
